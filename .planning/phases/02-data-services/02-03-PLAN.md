Census Geocoder API reference (from RESEARCH.md):
- Base URL: https://geocoding.geo.census.gov/geocoder
- Auth: none
- GET /geographies/address?street={street}&city={city}&state={state}&benchmark=Public_AR_Current&vintage=Current_Current&format=json
- Response path to lat/lon: result.addressMatches[0].coordinates.x (lon), .y (lat)
- Response path to FIPS: result.addressMatches[0].geographies["Census Tracts"][0].GEOID (11-digit)
- Empty addressMatches array = no match (new construction, etc.) -- must NOT crash

Census ACS API reference (from RESEARCH.md):
- Base URL: https://api.census.gov/data
- Auth: key={CENSUS_API_KEY} query param (free, from census.gov)
- GET /2023/acs/acs5?get=B19013_001E,B25002_001E,B25002_002E,B25002_003E,B23025_002E,B23025_005E,B01003_001E&for=tract:{tract}&in=state:{state}%20county:{county}&key={key}
- Variables: B19013_001E=median income, B25002_001E=total housing, B25002_002E=occupied, B25002_003E=vacant, B23025_002E=labor force, B23025_005E=unemployed, B01003_001E=population
- Response is 2D array: first row = variable names, second row = values
- state/county/tract come from Census Geocoder GEOID (11-digit: 2=state, 3=county, 6=tract)

Walk Score API reference (from RESEARCH.md):
- Base URL: https://api.walkscore.com
- Auth: wsapikey={WALKSCORE_API_KEY} query param
- GET /score?format=json&address={address}&lat={lat}&lon={lon}&transit=1&bike=1&wsapikey={key}
- Response: { walkscore, transit: {score}, bike: {score}, status }
- status 41 = daily quota exceeded (handle gracefully, don't crash)
- Requires lat + lon -- must run Census Geocoder first
- Permanent cache TTL (36500 days): same address never changes walkability meaningfully

    ```typescript
    import { z } from 'zod';

    // Census Geocoder raw response schema
    export const CensusGeocodeResponseSchema = z.object({
      result: z.object({
        addressMatches: z.array(z.object({
          matchedAddress: z.string(),
          coordinates: z.object({
            x: z.number(),   // longitude
            y: z.number(),   // latitude
          }),
          geographies: z.object({
            'Census Tracts': z.array(z.object({
              GEOID: z.string(),   // 11-digit state+county+tract
              TRACT: z.string(),
              STATE: z.string(),
              COUNTY: z.string(),
            }).passthrough()).optional(),
          }).passthrough().optional(),
        }).passthrough()),
      }),
    });

    // Normalized geocode result (what callers receive after parsing)
    export const GeocodeResultSchema = z.object({
      lat: z.number(),
      lon: z.number(),
      geoid: z.string(),     // 11-digit FIPS
      tract: z.string(),     // 6-digit tract
      state: z.string(),     // 2-digit state FIPS
      county: z.string(),    // 3-digit county FIPS
      matchedAddress: z.string(),
    });

    export type GeocodeResult = z.infer<typeof GeocodeResultSchema>;

    // Census ACS demographics result
    export const CensusACSResultSchema = z.object({
      medianIncome: z.number().nullable(),
      totalHousing: z.number().nullable(),
      occupiedHousing: z.number().nullable(),
      vacantHousing: z.number().nullable(),
      laborForce: z.number().nullable(),
      unemployed: z.number().nullable(),
      population: z.number().nullable(),
    });

    export type CensusACSResult = z.infer<typeof CensusACSResultSchema>;
    ```
