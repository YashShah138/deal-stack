    **Implementation details for underwriting.ts:**
    - Import Decimal from 'decimal.js'
    - Set `Decimal.set({ precision: 20, rounding: Decimal.ROUND_HALF_UP })` at module level
    - Implement each formula as a separate exported function matching RESEARCH.md patterns (lines 386-521)
    - Implement `runUnderwriting(input: UnderwritingInput): UnderwritingResult | FixerUpperResult`:
      - Calculate loan_amount = purchase_price * (1 - down_payment_pct/100)
      - Calculate all operating expenses from annual_gross_rent and percentage assumptions
      - Calculate NOI, cap rate, CoC, DSCR, GRM
      - Calculate equity projections at years 1, 3, 5, 10
      - If renovation_cost > 0 AND arv is provided, return FixerUpperResult with both scenarios
      - Post-reno scenario uses arv as the appreciated base, total_cash_invested includes renovation_cost

    **Implementation details for deal-score.ts:**
    - Weights: CoC 0.25, cap_rate 0.20, five_year_equity 0.20, market 0.15, value_add 0.10, comp 0.10
    - score = sum of (input * weight) for each component, clamped to 0-100
    - verdict: score >= 70 -> 'GO', score >= 50 -> 'CAUTIOUS GO', score < 50 -> 'NO'
    - Use decimal.js for the weighted sum calculation
